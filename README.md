## 설명
할 일과 완료해야할 날짜를 입력합니다. 
그리고 저장하게되면 할 일을 입력한 시점도 저장해 그 날부터 완료해야할 날짜를 계산해 기준으로 잡고 시간이 지남에 따라 퍼센트가 기준에 따라 증가하게됩니다.

할 일과 기간 그리고 진행도(시간 흐름)가 표시되고 X를 통해 삭제도 가능합니다.
